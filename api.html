API Reference

Detailed command-line and internal module documentation.

Command Line Interface

ghost train_rssp

Full GHOST pipeline: SSM pretraining → RSSP tree construction → per-node forest training.

FlagDefaultDescription
--data-Path to hyperspectral .mat cube (H, W, C).
--depthautoRecursive depth. 'auto' stops when SAM < 0.05.
--routinghybridModes: hybrid, forest, soft.
--forests5Parallel ensemble members per node.

ghost predict

Loads a trained bundle and runs cascade inference on test data.

FlagDescription
--modelPath to rssp_models.pkl.
--routingMode to use (or 'all' for comparative results).

Python API: Datasets & Models

HyperspectralDataset

Universal .mat loader with stratified spatial splitting.

ParameterRole
train_ratioFraction of labeled pixels for training (default 0.2).
use_fp16Enables min-max normalization for half-precision safety.

HyperspectralNet

The core GHOST model: CR → 3D Stack → SE → 2D U-Net.

ParameterDescription
num_filters3D spatial-spectral feature width (default 8).
base_filtersU-Net encoder starting capacity (default 32).

Python API: RSSP & SSSR

build_rssp_tree()

Constructs the binary spectral hierarchy from class mean statistics.

run_inference()

Executes soft path weighting and accumulates leaf probabilities.
