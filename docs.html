GHOST Architecture

Deep Reference: Design, Mathematics, and Implementation

What Problem Are We Solving?

A regular camera captures three numbers per pixel. A hyperspectral camera captures hundreds, creating a unique "spectral fingerprint" for every material. GHOST labels every pixel with its correct material class while remaining data-agnostic—it discovers band counts, class counts, and topologies at runtime.

1. Continuum Removal

Normalizes away brightness and albedo to isolate intrinsic absorption features. Enforced as a differentiable nn.Module at the front of every forward pass.

# s_min, s_max are per-pixel band extremes continuum(λ) = s_min + (s_max - s_min) × (λ / (C - 1)) CR(λ) = s(λ) / (continuum(λ) + ε)

2. Spectral 3D Convolution Stack

Treats the channel dimension as volumetric depth. Models local spectral correlations explicitly using a spectral kernel size of 7, coupling it with 3x3 spatial context.

Input (B, C, H, W) → Reshape (B, 1, C, H, W) → Conv3d(7, 3, 3)

3. Squeeze-and-Excitation (SE) Block

Learned channel attention. After the 3D stack produces over-complete features, the SE block recalibrates them by asking: "which channels matter for this particular image?"

4. 2D U-Net Encoder / Decoder

Progression through 32 → 64 → 128 → 256 → 512 channels. Captures multi-scale spatial context while preserving fine detail via skip connections. Bilinear interpolation ensures support for any input dimension.

5. RSSP — Recursive Spectral Splitting

Builds a binary hierarchy using Spectral Angle Mapper (SAM) distance. Spectrally distant classes are separated early, while similar ones are handled by specialist sub-models.

SAM(μ_i, μ_j) = arccos( (μ_i · μ_j) / (||μ_i|| × ||μ_j||) )

6. SSSR — Selective Spectral State Routing

Replaces hard classification cascades with soft, probabilistic routing. A multi-scale 1D-CNN (narrow/mid/wide kernels) produces pixel fingerprints that guide the dynamic routing.

7. Training Strategy

Uses AdamW with a ReduceLROnPlateau scheduler. Validation is performed every 20 epochs, selecting the best model based on mIoU to ensure sensitivity to minority classes.
